# City of Houston Capital Improvement Project map

I wanted to know more about capital improvement projects in Houston, so I mapped them by location and budget.

Source data from the City of Houston Open Data Portal:

2016-2020 Adopted Capital Improvement Plan - http://data.houstontx.gov/dataset/city-of-houston-capital-improvement-plan/resource/31efe2ef-6cc5-4db4-bd25-aff75842fb9e

Initial data prep involved eyeballing addresses and intersections, editing location data, and feeding it through the Google API to fetch lat/lon.
